# Mnookin Two-Pager (Responsive HTML)

A responsive HTML template for the Mnookin Two-Pager (from *Never Search Alone*). It centers content horizontally/vertically on desktop, scales to full width on mobile, and is print‑friendly.

## ✨ Features
- Emoji‑labeled sections (❤️ Love, 😤 Hate, ✅ Must‑Haves, 🚫 Must‑Nots, 🎯 Career Goals, 🌟 Candidate–Market Fit)
- Responsive design: centered card on desktop, full width on mobile
- Print‑friendly styling for easy PDF export
- Simple single‑file HTML + CSS, no dependencies

## 📂 Structure
- `index.html` – main HTML file with styles embedded

## 🚀 Usage
1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/mnookin-two-pager.git
   ```
2. Open `index.html` in your browser.
3. Edit text directly in the HTML to customize your Two‑Pager.

## 🖨 Printing
Use your browser’s **Print to PDF** option for a neat, shareable 2‑page document.
